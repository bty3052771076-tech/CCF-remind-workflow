## 📖 功能说明

### 1. 筛选功能

#### CCF等级筛选
- ☑ **A类** - 顶级会议和期刊（红色显示）
- ☑ **B类** - 重要会议和期刊（蓝色显示）
- ☑ **C类** - 一般会议和期刊（绿色显示）

#### 研究领域筛选
- 支持多选
- 包含15+研究领域
  - 人工智能、机器学习、数据库
  - 计算机网络、安全、软件工程
  - 计算机视觉、自然语言处理等

#### 搜索功能
- 实时搜索会议名称
- 搜索会议简介
- 支持中英文关键词

#### 天数范围
- 可调整查询范围（7-90天）
- 默认查询未来30天

---

### 2. 列表显示

#### 颜色标识

| 背景色 | 含义 | 剩余天数 |
|-------|------|---------|
| 🔴 红色 | 紧急 | ≤ 7天 |
| 🟠 橙色 | 需关注 | 8-15天 |
| 🔵 蓝色 | 即将到来 | 16-30天 |

#### 字体样式
- **A类** - 红色粗体
- **B类** - 蓝色粗体
- **C类** - 绿色粗体

#### 排序
- 按截止日期升序排列
- 最紧急的会议显示在最前面

---

### 3. 详情面板

点击列表中的任意会议/期刊，右侧详情面板会显示：

#### 基本信息
- 会议名称
- CCF等级
- 截止日期
- 剩余天数（带紧急程度标识）

#### 扩展信息
- 会议举办日期
- 官方网站链接
- 会议简介
- 研究领域列表
- 影响因子（期刊）

---

### 4. 工具栏功能

#### 🔄 刷新
重新加载数据文件，获取最新数据。

#### 📧 发送邮件
将筛选后的会议列表通过邮件发送。

**操作流程**:
1. 筛选需要的会议
2. 点击"📧 发送邮件"
3. 确认发送
4. 自动发送到配置的收件人

#### 💬 发送飞书
将筛选后的会议列表发送到飞书群。

**操作流程**:
1. 筛选需要的会议
2. 点击"💬 发送飞书"
3. 确认发送
4. 推送到飞书群聊

#### 数据切换
- **○ 会议** - 显示会议数据
- **○ 期刊** - 显示期刊数据

#### 天数调整
调整查询天数范围（7-90天）。

#### 🔍 应用筛选
应用当前筛选条件，更新列表。

## 💡 使用场景

### 场景1：查找AI领域的A类会议

**步骤**:
1. 启动GUI: `python gui_launcher.py`
2. 在左侧筛选面板勾选"A类"
3. 在研究领域列表中选择"人工智能"
4. 点击"🔍 应用筛选"
5. 查看筛选结果

**结果**: 显示所有AI领域的A类会议

---

### 场景2：查找即将截止的会议（7天内）

**步骤**:
1. 修改天数输入框为"7"
2. 点击"🔍 应用筛选"
3. 查看红色背景的紧急会议

**结果**: 显示7天内即将截止的所有会议

---

### 场景3：发送邮件提醒给团队

**步骤**:
1. 筛选需要提醒的会议
2. 点击"📧 发送邮件"
3. 确认发送

**结果**: 邮件自动发送到配置的收件人列表

---

### 场景4：搜索特定会议

**步骤**:
1. 在搜索框输入会议名称（如"CVPR"）
2. 列表自动更新，显示匹配结果

**结果**: 显示所有包含"CVPR"的会议

---

### 场景5：对比不同等级会议

**步骤**:
1. 只勾选"A类"，应用筛选
2. 查看A类会议数量和详情
3. 切换到"C类"，应用筛选
4. 对比查看

**结果**: 快速对比不同等级的会议

## 🔧 配置文件

GUI使用与命令行工具相同的配置文件：

- `config.json` - 邮件配置
- `feishu_config.json` - 飞书配置
- `conferences.json` - 会议数据
- `journals.json` - 期刊数据

**注意**: 修改配置文件后，点击"🔄 刷新"按钮重新加载。

## 📊 数据导出

### 导出为文本文件

1. 点击菜单: 文件 → 导出列表
2. 选择保存位置
3. 选择文件类型: `.txt`
4. 点击保存

**格式**:
```
CCF会议/期刊列表 - 2026-01-20
============================================================

名称: ICML 2026
等级: CCF-A
截止: 2026-01-28 (8天)
官网: https://icml.cc/

名称: VLDB 2026
等级: CCF-A
截止: 2026-02-01 (12天)
官网: https://vldb.org/
```

### 导出为JSON文件

1. 点击菜单: 文件 → 导出列表
2. 选择保存位置
3. 选择文件类型: `.json`
4. 点击保存

**格式**:
```json
[
  {
    "name": "ICML 2026",
    "rank": "A",
    "deadline": "2026-01-28",
    "days_until": 8,
    ...
  }
]
```

## 🐛 常见问题

### Q1: GUI启动失败？

**A**: 检查Python版本（需要3.7+）和tkinter是否安装：
```bash
python --version
python -m tkinter
```

### Q2: 数据无法加载？

**A**: 确认数据文件存在：
- `conferences.json` - 会议数据
- `journals.json` - 期刊数据

### Q3: 邮件发送失败？

**A**: 检查 `config.json` 配置：
- SMTP服务器地址
- 端口号（587/465）
- 用户名和授权码
- 收件人列表

### Q4: 飞书通知失败？

**A**: 检查 `feishu_config.json` 配置：
- Webhook URL是否正确
- 飞书机器人是否启用

### Q5: 筛选结果为空？

**A**: 检查筛选条件：
- 降低天数范围
- 勾选更多CCF等级
- 清空领域筛选
- 清空搜索关键词
